# Pokemon Tournament
A tool for Pokemon tournament.

## Dependency
Django 1.8

## Develop

### Windows
```shell
$ cd tools

// Prepare
$ pip install -r requirement.txt
$ init

// Start
$ runserver
```

### MAC / Linux
```shell
$ cd tools

// Prepare
$ sudo pip install -r requirement.txt
$ sudo sh init.sh

// Start
$ sh runserver.sh
```
