# Pokemon Tournament
A tool for Pokemon tournament.

## Dependency
Django 1.8

## Develop

### Start by command
```shell
// Prepare
$ sudo pip install -r requirement.txt
$ python manage.py migrate --settings=mysite.settings_default

// Start
$ python manage.py runserver --settings=mysite.settings_default
```

### Start by shell
```shell
// Prepare
$ sh setup.sh

// Start
$ sh start.sh
```
